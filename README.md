# Supplier Performance & Classification Dashboard

## Overview

This Power BI project was developed to analyze supplier performance, profitability, and supplier classification for a retail cooperative business.

The dashboard helps stakeholders identify high-performing suppliers, monitor profitability trends, classify suppliers based on performance metrics, and support data-driven procurement decisions.

---

## Business Problem

Managing supplier relationships requires visibility into:

* Supplier sales performance
* Profit contribution
* Supplier classification
* Geographic supplier distribution
* Trends over time

Traditional reporting made it difficult to quickly identify top-performing and underperforming suppliers.

---

## Project Objective

The objective of this project was to build an interactive Power BI dashboard that enables users to:

* Monitor supplier performance
* Analyze profitability
* Classify suppliers based on business metrics
* Support procurement and supply chain decisions
